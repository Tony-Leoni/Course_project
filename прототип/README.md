# 🚗 Прототип "Домклик Машино-места"

Интерактивный прототип платформы для безопасной покупки/продажи машино-мест с использованием сервиса безопасных расчётов (СБР) Сбербанка.

## 🎯 Особенности

✅ **Полный цикл сделки** — от поиска до регистрации в Росреестре  
✅ **Безопасные расчёты** — интеграция с СБР Сбербанка  
✅ **Личный кабинет** — просмотр купленных объектов  
✅ **Современный UX** — плавные анимации и микроинтеракции  
✅ **Мобильная оптимизация** — адаптивный дизайн  

## 🚀 Быстрый старт

1. Откройте файл `app/index.html` в браузере
2. Выберите сценарий: "Покупатель" или "Продавец"
3. Следуйте пользовательскому пути

## 🎬 Пользовательские пути

### 🔵 Сценарий "Покупатель"
1. **Поиск** → Каталог машино-мест на портале
2. **Просмотр** → Детали объекта с фото и параметрами
3. **Авторизация** → Вход через Сбер ID
4. **Эскроу** → Заморозка средств на номинальном счёте
5. **Подписание** → ДКП через Госключ (УКЭП)
6. **Трекер** → Отслеживание регистрации в Росреестре
7. **Личный кабинет** → Просмотр купленного объекта

### 🟢 Сценарий "Продавец"
1. **Инициация** → Ввод кадастра и параметров сделки
2. **Расчёт комиссии** → Автоматический подбор оптимального тарифа
3. **Генерация ссылки** → P2P-ссылка для отправки покупателю
4. Далее переход к шагам покупателя (4-7)

## 🎯 API Интеграции (концепт)

Прототип демонстрирует интеграции с:
- ✅ **Сбер ID** — авторизация
- ✅ **СБР (Сервис Безопасных Расчётов)** — эскроу
- ✅ **Госключ** — электронная подпись (УКЭП)
- ✅ **Росреестр Fast Track API** — ускоренная регистрация
- ✅ **ЕГРН** — проверка кадастровых данных
