# Продуктовая Стратегия: «Домклик — Машино-места»

Ребят, я обновил нашу презентацию в HTML‑формате — это версия, которую будем показывать Масленникову.

## Как открыть презентацию
1. **Скачайте ZIP‑архив** репозитория (кнопка Code -> Download ZIP).
2. **Распакуйте его** у себя на компьютере.
3. **Откройте основной HTML‑файл** (например, `1_Market_Analysis.html`) в браузере — все вкладки презентации будут доступны на боковой панели.
